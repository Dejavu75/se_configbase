# se_configbase

Paquete Node/TypeScript compartido para configuración base del ecosistema. Centraliza controladores, esquemas y utilidades comunes usadas por servicios Node.

## Camino rápido

```bash
npm install
npm run tsc
```

Ejecución del build publicado:

```bash
npm start
```
